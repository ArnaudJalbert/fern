# Welcome to the Vault

This is a sample vault for testing **Fern**. It contains:

- **Inbox** — a database for quick captures
- **Projects** — a folder with two project databases inside
- **Reading List** — a database for tracking books and articles
- **Journal** — a folder with loose daily notes

Feel free to explore, edit pages, and create new databases.
