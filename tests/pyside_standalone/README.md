# Standalone test vault

Fake data for testing the Fern UI. Open this folder as a vault via **Vault > Open...** (or from the welcome page).

Structure:

- **Databases/Notes/** — Quick note, Ideas
- **Databases/Projects/** — Project Alpha, Project Beta
- **Databases/Inbox/** — Inbox

Each `.md` file has YAML frontmatter with `id:` and body content.
